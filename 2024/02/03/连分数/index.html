连分数

强网青少赛 未解之谜

x =getRandomRange(2, int(root(n, 4))) // 8 d = phi -x ed = 1 mod phi ephi -ex = 1 mod phi -ex = 1 mod phi ex = -1 mod phi ex = kphi -1 e/phi = k/x - 1/xphi

其中n是2048bit量级，而x是509bit量级，因此上式成立，即符合勒让德理论，因此可以连分数展开求d 需要注意的是，我们求的是x，但实际解密需要用−x解

攻击原理

勒让德定理：对于正有理数f,如果整数a和正整数b，满足|f-a/b|<1/2b^{2} 那么a/b等于f的连分数的一个收敛子

仔细观察上式，因为phi本身很大，而且gcd(k,d)=1，k/d与e/phi非常接近。

此外phi = (p-1)*(q-1)= N-(p+q)+1 其与N的差值也相对很小，所以k/d与e/N也相差不大。RSA的(N,e)都是公开的，由此维纳大胆地构想——如果对e/N进行连分数展开，有可能k/d就是其中的某个收敛子！
